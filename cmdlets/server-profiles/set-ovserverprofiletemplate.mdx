## Synopsis

Modifies an existing server profile template resource on an HPE OneView appliance.

## Syntax

```powershell
Set-OVServerProfileTemplate
    [-InputObject] <Object>
    [-ApplianceConnection <Object>]
    [-Async]
    [<CommonParameters>]
```

## Description

`Set-OVServerProfileTemplate` applies modifications to an existing server profile template object and saves the updated configuration back to the HPE OneView appliance.

The typical workflow is:

1. Retrieve the template with `Get-OVServerProfileTemplate`.
2. Modify the returned object's properties directly in PowerShell.
3. Pass the modified object to `Set-OVServerProfileTemplate`.

After the template is updated, derived server profiles that were previously compliant may become non-compliant. Use `Get-OVServerProfile -NonCompliant` to identify affected profiles and `Update-OVServerProfile` to remediate them.

<Warning>
`Set-OVServerProfileTemplate` sends the entire template object back to the appliance. Modifying a template that has derived profiles will affect the compliance state of those profiles. Review downstream impacts before saving changes.
</Warning>

## Parameters

<ParamField path="InputObject" type="object" required>
  The server profile template object to update, as returned by `Get-OVServerProfileTemplate`. Accepts pipeline input.
</ParamField>

<ParamField path="Async" type="switch">
  Return a task object immediately without waiting for the update to complete.
</ParamField>

<ParamField path="ApplianceConnection" type="object">
  Specifies the HPE OneView appliance connection. Defaults to the default connected session. Alias: `Appliance`.
</ParamField>

## Examples

### Update a template description

```powershell
$template = Get-OVServerProfileTemplate -Name 'Hypervisor Cluster Node Template v1'
$template.description = 'Updated: Corp standard hypervisor cluster node, version 1.1'
Set-OVServerProfileTemplate -InputObject $template | Wait-OVTaskComplete
```

### Update a template via pipeline and return immediately

```powershell
$template = Get-OVServerProfileTemplate -Name 'Production Node Template v1'
$template.serverProfileDescription = 'Auto-generated by provisioning pipeline'
$task = $template | Set-OVServerProfileTemplate -Async
Write-Host "Update task: $($task.uri)"
```

## Output

`HPEOneView.Appliance.TaskResource` — An async task object tracking the update operation.
